# 💳 Pembayaran Digital Kasir

Sistem pembayaran digital sederhana berbasis web yang dirancang untuk membantu proses transaksi pada kasir, cocok untuk digunakan di toko kecil, warung, atau sebagai latihan proyek pembelajaran.

---

## 📌 Deskripsi

Proyek ini mensimulasikan proses kasir dalam melakukan:
- Input barang belanjaan (nama, harga, jumlah)
- Perhitungan total belanja
- Proses pembayaran dan kembalian

Antarmukanya dibuat sederhana agar mudah digunakan, baik oleh pengguna maupun pengembang pemula.

---

## ✅ Fitur Utama

- Menambahkan item belanjaan
- Menghitung total harga otomatis
- Input nominal pembayaran
- Menampilkan kembalian
- Antarmuka bersih dan ringan
- *(Opsional)* Simpan riwayat transaksi

---

## 🛠️ Teknologi yang Digunakan

- HTML & CSS → Tampilan antarmuka
- JavaScript → Logika transaksi dan interaksi
- *(Opsional)* PHP & MySQL → Penyimpanan riwayat transaksi
